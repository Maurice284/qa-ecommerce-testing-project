# QA E-commerce Testing Project

This project demonstrates manual QA testing for an e-commerce web application similar to Amazon. It includes a complete testing workflow: creating a test plan, writing test cases, executing tests, and logging bugs. The goal is to showcase practical QA skills and understanding of standard testing practices.

## Key Functionalities

The project focuses on key functionalities:

- User registration and login
- Product search and filtering
- Product detail validation
- Add to cart and cart management
- Checkout process
- Coupon and payment validation
